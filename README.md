Password Complexity Checker

The Password Complexity Checker is a web-based tool that evaluates the strength of a password based on different security criteria. It checks factors such as password length, presence of uppercase and lowercase letters, numbers, and special characters. The tool provides real-time feedback to help users create stronger and more secure passwords.

FEATURES: 

●Checks minimum password length

●Detects uppercase and lowercase letters

●Verifies numbers and special characters

●Displays password strength (Weak / Medium / Strong)

●Provides real-time feedback

●Simple and user-friendly interface
